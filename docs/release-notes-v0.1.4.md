## ✨ 新功能

- **外观设置**：设置页新增「外观」分区，三项均即时生效并保存到本地——
  - **明暗模式**：深色 / 浅色 / 跟随系统。浅色为完整重绘配色，状态圆点与运行徽标保持鲜亮醒目。
  - **贴纸不透明度**：60%–100% 自由调节桌面贴纸的背景透明度（仅作用贴纸，文字始终清晰）。
  - **界面密度**：紧凑 / 标准 / 宽松，一键调整贴纸卡片的字号与间距。
- **Context 百分比更准 + 零配置接入**：上下文已用百分比改为直接取自 Claude Code 的 statusline——1M 上下文窗口也能正确显示，不再一律顶到 100%；安装后自动接入 CC 设置（hooks + statusLine），无需手动配置。

## 💅 体验优化

- 用量重置时间精确到钟点：今天显示剩余时长，明天 / 后天 / 具体日期均带具体时刻。
