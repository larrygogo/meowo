## ✨ 新功能

- **会话错误检测**：会话因「工具调用解析失败 / 需要重新登录 / 认证失败」而卡死时，卡片转红并归入「待交互」，一眼可辨。
- **桌面通知**：会话需要你回复（待交互）或出错时，弹一条去重的系统通知（同一情形只弹一次）；**点击通知直接切到该会话的终端**。可在设置「通用」里用总开关统一控制，默认开启。
- **账号与用量**：设置页新增「账号」分区——显示当前 Claude Code 账号、实时配额（5 小时 / 7 天 用量百分比与重置时间、各模型用量）、以及每日用量热力图（GitHub 贡献图风格）。登录 token 过期自动续期；未登录有清晰引导。
- **托盘左键打开设置**：左键单击托盘图标直接打开设置窗口，右键仍为菜单。

## 💅 体验优化

- 自定义滚动条样式（细、半透明、配深色主题）。
- 设置页各分区视觉统一（标题并入卡片）、卡片留白与边框更清晰。

Windows 用户下载 `cc-kanban_0.2.0_x64-setup.exe` 安装即可；已安装旧版可在应用内 / 托盘「检查更新」自动升级到本版。
